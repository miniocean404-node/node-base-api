# 权限修改

```js
// 异步地更改文件的权限。 除了可能的异常之外，没有为完成回调提供任何参数。
fs.chmod("./1.txt", 0o755, (err) => {});

// 异步地更改文件的所有者和群组。 除了可能的异常之外，没有为完成回调提供任何参数
fs.chown("./1.txt", 1, 1, (err) => {});

// 通过 fd 修改权限
fs.fchmod(1, 0o755, (err) => {});
// 更改文件的所有者和群组
fs.fchown(1, 1, 1, (err) => {});
```

# 文件系统时间戳
```js
// 更改 path 引用的对象的文件系统时间戳。
fs.utimes("1.txt", Date.now(), Date.now(), (err) => {});
// 以与 fs.utimes() 相同的方式更改文件的访问和修改时间，不同之处在于，如果路径引用符号链接，则不会取消引用该链接：而是，符号链接本身的时间戳会更改。
fs.lutimes("1.txt", Date.now(), Date.now(), (err) => {});
// fd 更改 path 引用的对象的文件系统时间戳。
fs.futimes(1, Date.now(), Date.now(), (err) => {});
```
